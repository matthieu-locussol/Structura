Detailed Description

template<typename T>
class St::Stack< T >

Datatypes class that represents a Stack data structure.

A stack is a data structure used to store a collection of objects. Individual items can be added and stored in a stack using a push operation. Objects can be retrieved using a pop operation, which removes an item from the stack.

◆ rotateLeft()

template<typename T >
void St::Stack< T >::rotateLeft

The rotateLeft method will rotate the stack by moving the last element to the top position. The top element will then become the second, the second the third, and so on. If the stack contains less than 2 elements, it leaves it unchanged.

template<typename T >
void St::Stack< T >::rotateRight

The rotateRight method will rotate the stack by moving the top element to the last position. The second element will become the first, the third will become the second, and so on. If the stack contains less than 2 elements, it leaves it unchanged.
